# 타입의 목적

폰 노이만 구조를 가지는 컴퓨터의 메모리는 단순히 거대한 바이트 배열로 표현할 수 있습니다.
CPU는 배열이나 구조체 등 프로그램이 정의한 타입에 대해 알지 못합니다. 
프로그래밍 언어의 타입이 CPU가 메모리를 해석할 때 무슨 기능을 하는지 고르시오. 

- [ ] 메모리에 저장된 바이트의 순서를 바꿔 프로그램의 실행 속도를 가속함.
- [x] 컴파일러와 개발자에게 특정 메모리를 일관되게 해석하는 방법을 알려줌.
- [ ] 운영체제가 타입에 알맞는 메모리를 할당하도록 유도함.
- [ ] CPU가 불변속성(Invariant)를 지키지 않는 값을 감지하도록 도와줌.
