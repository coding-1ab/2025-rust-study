# if의 결과값

아래 세 가지 코드 예시중 올바른 것을 고르시오.

```rs
fn main() {
    // A.
    let x = 5;
    let y = if x > 3 { "big" } else { "small" };

    // B.
    let x = 5;
    let y = if x > 3 { "big" };
        else { "small" };

    // C.
    let x = 5;
    let y = if x > 3 { "big" } else { 0 };
}
```

- [x] A.
- [ ] B.
- [ ] C.
- [ ] 없음.
